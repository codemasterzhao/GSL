Method Overview

We introduce Generalizable Hierarchical Skill Learning (GSL), a novel framework for hierarchical policy learning that significantly improves policy generalization and sample efficiency in robot manipulation. GSL uses object-centric skills as an interface that bridges high-level vision-language models and low-level visual-motor policies.

Object-Centric Skill Learning

  • High-level agent: Decomposes demonstrations into transferable skill primitives using foundation models and selects skill-object pairs for task execution.
  • Low-level policy: Learns canonical skills in object-centric coordinate frames, enabling generalization across object poses and appearances.

This structured yet flexible design enables the high-level to convey semantic intent while the low-level performs localized control without reasoning over the entire scene.

Mask-as-Frame Interface

Object masks serve as spatial reference frames, canonicalizing skills in object coordinates:

  • Transferable skills: Object-centric canonicalization makes skills reusable across different object poses and instances.
  • Exceptional efficiency: Trained with only 3 demonstrations per task outperforms baselines trained with 100 demonstrations.

Experiments

Our method is tested on GemBench simulation with 10 training tasks and 22 testing tasks, and validated on a real robot across 6 real-world manipulation tasks. GSL demonstrates superior generalization across spatial arrangements, object appearances, and task compositions. Check our paper for detailed results.
